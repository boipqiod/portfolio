회고

Electron으로 리팩토링

기존에는 SaaS으로 개발하기 위해 Express를 사용해 개발했습니다.
Express를 이용해 서버에서 Youtube API를 이용한 채팅 읽기, 채팅 전송 기능을 구현했습니다.
하지만, Youtube API는 각 사용자 마다 API Key를 받아야되고, 수량도 한정적이라는 점 등 한계로 인해 Electron으로 리팩토링을 진행했습니다.
이를 통해 Youtube API를 사용하지 않고, Electron 창의 DOM 조작을 통해 채팅 읽기, 채팅 전송 기능을 구현할 수 있었습니다.
